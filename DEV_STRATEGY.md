# 🚀 Predata 개발 전략

## ✅ **현재 권장 방식: Mock 모드**

Web3 통합 코드는 모두 준비되었지만, **로컬 개발 시에는 비활성화**하고 진행합니다.

### 왜 이 방식이 좋은가?

1. ✅ **빠른 개발** - 블록체인 설정 불필요
2. ✅ **완전 무료** - 가스비 걱정 없음
3. ✅ **간단한 테스트** - DB만으로 모든 기능 테스트
4. ✅ **안정적** - Node.js 버전 문제 없음

### 현재 시스템

```
사용자 → 프론트엔드
           ↓
      Spring Boot
           ↓
    MariaDB 저장 ✅
           ↓
    블록체인? ❌ (비활성화)
```

**모든 데이터가 MariaDB에 저장되어 완전히 작동합니다!**

---

## 🎯 **나중에 Web3 활성화**

### 서비스가 완성되고 실제 배포할 때:

1. **Base Sepolia 테스트넷에 배포**
   ```bash
   cd blockchain
   npx hardhat run scripts/deploy.js --network baseSepolia
   ```

2. **설정 활성화**
   ```yaml
   blockchain:
     enabled: true
     contract:
       address: "0x..."  # 배포된 주소
   ```

3. **끝!** 코드 변경 없이 Web3 기능 활성화
